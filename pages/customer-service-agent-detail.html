智能客服 Agent

基于大语言模型的多轮对话智能客服系统，集成 RAG 知识库检索、意图识别与情感分析，支持企业微信、钉钉等多平台接入，实现 7×24 小时自动客服。

LLM 驱动 RAG 检索 多平台接入 7×24 在线

产品概述

本系统是一套面向企业级场景的智能客服解决方案。以 DeepSeek / GPT-4o-mini 等大语言模型为核心引擎，结合向量检索（RAG）实现精准知识问答，辅以意图识别和情感分析模块，能够理解用户真实意图、感知情绪变化，提供自然流畅的多轮对话体验。


适用场景涵盖：售后咨询（退换货、物流、保修）、产品使用指导、业务办理引导、内部 IT 服务台等。通过统一的 Message Router 可无缝接入企业微信、钉钉、Web Chat 等渠道。

核心能力

🧠 意图识别引擎

基于 LLM 的 Few-shot 意图分类，支持 20+ 业务意图的精准识别。结合置信度阈值与降级策略，低置信度场景自动转人工或引导用户澄清，保障对话质量。

📚 RAG 知识检索

Embedding + 向量相似度搜索，从 FAQ / 产品手册中精准召回相关知识片段，与 LLM 生成结合输出有据可依的答案。支持关键词 + 语义双路召回，提升命中率。

💬 多轮对话记忆

会话级上下文管理，基于 TTL 的滑动窗口记忆机制。自动维护最近的 N 轮对话历史，支持跨轮指代消解和上下文连贯回复。

❤️ 情感分析

实时识别用户情绪（正面 / 中性 / 负面），当检测到强烈不满或投诉倾向时，自动触发安抚策略并优先转接人工坐席，避免舆情升级。

🛡️ 安全护栏

多层安全防护：Prompt 注入检测、PII 敏感信息脱敏、输出内容过滤、敏感词拦截。确保系统不被恶意利用，保障用户隐私与合规性。

🔌 多平台接入

统一的 Message Router 抽象层，通过 Adapter 模式接入企业微信、钉钉、Web Chat、API 等多种渠道。消息格式统一转换，业务逻辑零耦合。

技术栈

Python 3.10+ FastAPI / aiohttp LangChain DeepSeek / GPT-4o-mini PostgreSQL Redis Milvus Docker Kubernetes Pydantic Sentence-Transformers Prometheus + Grafana

LLM 网关支持 OpenAI 兼容 API，可灵活切换 DeepSeek / GPT-4o-mini / 本地模型。知识库 Embedding 采用 Sentence-Transformers，向量检索基于 Milvus。会话状态与缓存通过 Redis 管理，持久化数据存储于 PostgreSQL。

快速启动

Phase 1 已完成，可直接运行体验。需要 Python 3.10+ 和 DeepSeek API Key。

# 1. 进入项目目录 cd customer-service-agent # 2. 配置 API Key # 编辑 .env 文件，修改 OPENAI_API_KEY # 3. 安装依赖 pip install aiohttp langchain langchain-openai langchain-core pydantic pydantic-settings httpx python-dotenv tenacity # 4. 启动服务 python -m app.main # 5. 打开浏览器访问 # http://localhost:8000 → 网页聊天界面

📁 项目结构：app/api/（路由）、app/core/（LLM / 记忆 / 护栏 / 知识库）、app/models/（数据模型）、static/（Web 界面）、knowledge/（FAQ 文件）
