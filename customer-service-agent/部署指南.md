# 部署指南：上线到公网 + 绑定域名

## 方案对比

| 方案 | 费用 | 难度 | 适合场景 |
|------|------|------|----------|
| **Render** | 免费（512MB） | ⭐ 极简 | 个人项目、快速上线 |
| **Railway** | $5/月起 | ⭐ 简单 | 长期运行、较高性能 |
| **阿里云 ECS** | ~¥50/月 | ⭐⭐ 中等 | 国内访问快、备案域名 |

---

## 方案一：Render（推荐，免费 + 最简单）

### 1. 注册 Render
打开 https://render.com → 用 GitHub 账号注册

### 2. 部署
- 点 `New +` → `Web Service`
- 连接 GitHub 仓库（先把项目 push 上去）
- Render 自动识别 `render.yaml`，点击创建即可
- 在 Environment 里设置 `OPENAI_API_KEY` 为你的 DeepSeek Key

### 3. 绑定域名
- 在 Render Dashboard → Settings → Custom Domain
- 添加你的域名，Render 会给你一个 CNAME 记录
- 去域名 DNS 管理页添加 CNAME 解析，等待生效

### 4. 如果有 GitHub 仓库

```bash
cd customer-service-agent
git init
git add .
git commit -m "init: customer service agent"
git branch -M main
git remote add origin https://github.com/你的用户名/repo名.git
git push -u origin main
```

## 方案二：Railway（免费额度）

### 1. 注册 Railway
https://railway.app → GitHub 登录

### 2. 部署
- `New Project` → `Deploy from GitHub repo`
- 选择仓库，Railway 自动检测 Procfile
- 在 Variables 里添加 `OPENAI_API_KEY`

### 3. 绑定域名
- Settings → Networking → Custom Domain
- 添加域名，配置 CNAME 记录

---

## 方案三：阿里云轻量服务器（国内用户推荐）

### 1. 购买
- 阿里云轻量应用服务器，最低 ~¥50/月
- 选择 CentOS 7.9 或 Ubuntu 22.04

### 2. 连接服务器
```bash
ssh root@你的服务器IP
```

### 3. 安装 Docker
```bash
curl -fsSL https://get.docker.com | sh
```

### 4. 部署
```bash
# 在服务器上
mkdir -p /app/cs-agent
cd /app/cs-agent

# 把项目文件上传（用 scp 或 git clone）
# 然后运行：
docker build -t cs-agent .
docker run -d --name cs-agent --restart=always \
  -e OPENAI_API_KEY=你的Key \
  -p 8000:8000 \
  cs-agent
```

### 5. 安装 Nginx 反向代理 + HTTPS
```bash
apt install nginx certbot python3-certbot-nginx -y

# 配置 Nginx
cat > /etc/nginx/sites-available/cs-agent << 'EOF'
server {
    listen 80;
    server_name 你的域名.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
EOF

ln -s /etc/nginx/sites-available/cs-agent /etc/nginx/sites-enabled/
nginx -t && systemctl reload nginx

# 申请免费 SSL 证书
certbot --nginx -d 你的域名.com
```
